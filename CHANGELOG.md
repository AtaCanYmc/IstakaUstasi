# Changelog

All notable changes to this project will be documented in this file. This project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

This log is automatically generated and updated by Google's `release-please` workflow.

---

## [1.0.0] - 2026-07-18
### Added
* Fully implemented Phase 3 Okey Solver Web App using React, TypeScript, and Vite.
* Unified global state store with Zustand.
* Interactive 40-slot rack board supporting drag-and-drop actions.
* Manual tile selector pool and Okey indicator tile configurations.
* Multi-stage frontend Docker integration with Nginx reverse proxy.
* Root Docker Compose orchestrations bridging frontend and backend.
* Github Actions CI/CD workflows for linting, typing, building, and automated releases.

## [0.1.0] - 2026-07-17
### Added
* Initial release of the FastAPI bridge microservice for solving Okey hand arrangements and scanning boards via computer vision.
